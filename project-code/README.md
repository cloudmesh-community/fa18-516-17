## Setup

* Linux 64-bit system
* 8mb of RAM
* 20 GB hardrive space
* Ubuntu-18.04.1-desktop-amd64.iso(1.82GB)
* Computername:516
* Username:student

```bash
sudo apt-get install git
mkdir project
cd project
git clone https://github.com/cloudmesh-community/fa18-516-17.git
```

```bash
Restart the virtualbox
# Hadoop  Installation
```
